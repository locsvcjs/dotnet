# dotnet commands

Raspberry Pi Installer

```console
sudo apt-get update -y

wget -O - https://raw.githubusercontent.com/locsvcjs/dotnet/main/install.sh | sudo bash
sudo reboot
dotnet new console -o console1
cd console1
dotnet run
```
